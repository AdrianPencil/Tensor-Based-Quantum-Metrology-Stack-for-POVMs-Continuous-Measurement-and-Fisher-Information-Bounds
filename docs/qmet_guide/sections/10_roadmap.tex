\section{Roadmap}

Planned extensions are organized by preserving the current module boundaries.

\subsection{Sensor layer}

\begin{itemize}
  \item multi-parameter Hamiltonians and control pulses
  \item additional decoherence channels beyond pure dephasing
\end{itemize}

\subsection{Measurement layer}

\begin{itemize}
  \item richer POVMs and projective measurement libraries
  \item more readout noise families (Poisson counting, colored noise)
\end{itemize}

\subsection{Estimation layer}

\begin{itemize}
  \item multi-parameter Bayesian updates
  \item constrained estimators and priors
  \item simple optimizer-based MLE for small models
\end{itemize}

\subsection{Workflows and reporting}

\begin{itemize}
  \item reproducible experiment bundles (config in, artifacts out)
  \item auto-generated tables and plots for the guide
\end{itemize}
